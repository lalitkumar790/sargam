# 🎵 Sargam – Interactive Riyaaz & Music Learning Platform

Sargam is a **web-based Indian classical music learning and practice platform** designed to make *riyaaz (daily practice)* interactive, accessible, and engaging.
It combines **real-time audio playback, gamified learning, recording, and composition tools** into a single system.

---

##  Features

###  Interactive Sargam Instrument

 Play Sa Re Ga Ma Pa Dha Ni using keyboard keys
 Multiple instrument sounds
 Real-time audio playback using Web Audio API

---

###  Recording Studio

 Record your keystrokes with timing
 Replay performances
 Clear and re-record sessions

---

###  Raga Bubble Game

 Learn ragas through a visual gamified experience
 Notes fall as bubbles and play in sequence
 Improves pattern recognition and timing

---

###  Sargam Roll Composer

 Grid-based interface (time vs notes)
 Create and edit musical sequences
 Export compositions as JSON

---

###  Advanced Audio Engine

 Built using Web Audio API
 Features:

  * Volume control
  * Pitch shifting
  * Reverb effects
 Low latency and precise timing

---

##  Project Vision

Sargam is evolving into a Daily Riyaaz Companion that helps users:

 Practice ragas daily
 Improve note accuracy
 Track progress over time
 Learn Indian classical music interactively

---

##  Tech Stack

 *Frontend:* HTML, CSS, JavaScript
 *Audio Engine:* Web Audio API
 *Storage:* LocalStorage (for session data)
 *Architecture:* Modular JavaScript
